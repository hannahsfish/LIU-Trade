# LEI 投资分析框架 - 完整研究发现

## 概述
LEI 投资分析框架是一个将基本面（战略）与技术面（战术）深度融合的系统，强调"不预测，只预言"——利用数学逻辑推演市场的"必由之路"。

核心哲学源自《孙子兵法》：基本面解决"知可以战与不可以战者胜"（选股），技术面解决"善战者胜于易胜者也"（择时）。

---

## 第一部分：基本面分析 (Strategy: "What to Buy")

### 商业模式四大基本条件
5分钟内无法理解其生意模式，直接放弃。

1. **无差别产品** - 标准化程度高，易于扩张
2. **最广泛用户** - 潜在客户群庞大（如全球几十亿人）
3. **自由定价权** - 价格由公司决定，不受政府或非市场因素管制
4. **市场垄断** - 具有极深的护城河

### 财务筛选标准
| 指标 | 门槛 | 逻辑 |
|------|------|------|
| 年收入 | > $100M | 生存门槛，跨越后进入良性发展 |
| 收入增长 | 15-20% | 低于10%动力不足，高于40%难以维持，15-20%是黄金增速 |
| 运营现金流 | > 0 | 生意本身能赚钱 |
| 自由现金流 | > 0 | 可攻可守（研发/并购/分红/回购） |
| 运营利润率 | 越高越好 | 同业比较，坚决回避结构性亏损 |

**避雷针**：坚决回避结构性亏损（利润为负、债务>收入）的企业。

### 估值指标
- **P/S 市销率**、**P/E 市盈率** — 不作为买卖依据，仅用于了解市场预期
- 高估值 = 高预期，低估值 = 低预期

## 第二部分：技术分析 (Tactics: "When to Buy")

### 时钟方向比喻
| 方向 | 含义 | 状态 |
|------|------|------|
| 12-1点 | 极度看好 | 不可持续的暴涨（协率极端） |
| 2点 | 最佳状态 | 基本面与股价共振，稳步上涨 |
| 3点 | 犹豫怀疑 | 横盘震荡 |
| 4点 | 不认可 | 开始下跌 |
| 5-6点 | 极度悲观 | 暴跌或深度下跌 |

**核心**：只做"2点钟方向"的股票

### 协率 (Slope) — 趋势的加速与减速

趋势不是直线，而是曲线，其强度变化体现为协率变化。

**判断加速**：上涨角度越来越陡峭。趋势从"开始阶段"（平缓）→"发展阶段"（真实协率）→"极端状态"（几乎垂直，接近尾声）。

**判断减速/转折**：使用均线量化协率。通过抵扣价判断：如果股价无法维持在抵扣价之上，均线变平或拐头向下 = 协率改变 = 趋势减速或反转。

### 具体均线周期

| 均线 | 类型 | 用途 |
|------|------|------|
| MA20 / EMA20 | 短期 | 短线拐头和买点判断 |
| MA60 / EMA60 | 中期 | 中期趋势和抵扣价推演 |
| EMA120 | 长期 | 大趋势判断、乖离率基准线 |

- 抵扣价原理基于 SMA
- 趋势验证和乖离率使用 EMA（特别是 120日EMA）
- 实操中 MA 和 EMA 双重验证

## 第三部分："价格/成交量/时间/空间"交易系统

### 1. 价格 (Price) - 趋势的本质是成本
- **均线 (MA/EMA)**：代表市场平均成本
- **协率 (Slope)**：均线的倾斜度，用均线量化
- **2B结构**：假跌破后迅速拉回，趋势转折信号

### 2. 时间 (Time) - 核心技术："抵扣价"

**数学原理**：
- 今天的N日均线 = (今天收盘价 + 前N-1天收盘价总和) / N
- 明天的N日均线 = (明天收盘价 + 前N-1天收盘价总和 - N天前的收盘价) / N
- **结论**：若要明天均线向上，必须满足：明天收盘价 > N天前的收盘价
- N天前的收盘价 = **抵扣价**

**"画框"推演法（精确步骤）**：
1. **确定周期**：假设判断 60日均线是否会拐头向上
2. **回溯历史**：找到 60天前 的股价位置
3. **寻找下跌窗口**：观察 60天前的股价走势。如果接下来一段时间（如未来10天），抵扣价从高位迅速下跌到低位
4. **画框推演**：在当前K线图右侧画框，覆盖对应时间跨度。在此时间框内，抵扣价（分母）从高位迅速变小
5. **设定买点**：计算出即将到来的"低位抵扣价"具体数值。只要当前股价能横盘或维持在此抵扣价之上，均线**必须**拐头向上。这个价格就是**"必由之路"**

### 3. 空间 (Space) - 利润空间
- **均线密集区 (MA Concentration)**：
  - 无硬性百分比定义，核心是"成本趋于一致"的逻辑判断
  - 最理想：20/60/120 三线合一
  - 次优：20+60 汇聚也可作为第一波起点（如GE案例）
  - 买点不是密集本身，而是放量突破 + 均线向上发散
  - 周线级别抓大行情，日线级别找切入点
  - 市场共识区域，突破后看向下一密集区
- **乖离率 (Bias Ratio)**：
  - 公式：|当前收盘价 - 120日EMA| / 120日EMA × 100%
  - 基准均线：**120日 EMA**
  - 极端值：50%-70%偏离 = "人性之极"，股价有强烈回归需求
- **盈亏比**：目标空间 / 止损空间，至少 2:1

### 4. 成交量 (Volume) - 分歧与共识
- 成交量大 = 分歧大（多空互搏）
- 成交量小 = 分歧小（交投清淡）
- **极端放量**：底部巨量 = 分歧产生 = 可能转折
- **筹码峰 (Volume Profile)**：
  - 时间范围是**动态选定的**，取决于观察的行情段落（一波上涨、震荡箱体等）
  - 统计该时间范围内每个价格档位上的成交量总和
  - 堆积最大的价格区间 = 市场公允价格
  - 具有最强支撑/阻力作用
  - **与均线交叉验证**：当股价回踩均线（如60日）且该位置恰好也是主要筹码峰时，支撑力度最强
  - 均线 = 固定周期平均成本，筹码峰 = 特定价格区间成交量堆积，两者互相验证
  - **时间范围选取逻辑**：
    - 找支撑 → 选前期盘整/震荡段（共识形成，筹码最厚）
    - 分析回撤 → 选底部启动到现在的完整趋势段（主力资金堆积位置）

## 第四部分：核心交易信号

### 2B结构（左侧交易/博弈性买入）

**定义**：下跌趋势中创出新低，但迅速拉回并突破前低点（空头陷阱）

**识别步骤**：
1. 找到下跌中的低点 A
2. 股价跌破 A 创出新低 B
3. 股价在 B 点迅速反弹，收盘价站上 A 点

**关键细节**：
- 收盘价必须实质性站回前低上方（盘中下影线刺破不算），价格重心实质收回
- "迅速"是定性视觉判断（"马上回来"），无硬性天数限制
- 验证有效性的核心：**抵扣价窗口**
  1. 2B结构出现时，立刻画框推演未来
  2. 计算未来X天/周内，20日均线抵扣价是否处于低位或迅速下移
  3. 2B结构必须在画框时间内稳住股价，熬过高位抵扣价，迫使20日均线拐头向上
  4. 价格止跌 + 均线拐头 = 两者共振才是有效买点

**仓位**：确定性较低，属于博弈性质，**轻仓试探**

### 均线密集（右侧交易/确定性重仓）

**定义**：20日、60日、120日三条均线汇聚在狭窄价格区间内

**识别**：
- 三条均线"纠缠"在一起，不再发散
- 含义：短中长期持仓成本趋于一致，市场共识形成
- 突破后均线向上发散 + 放量 = 大趋势确立

**级别区分**：
- **周线级别密集** = 大行情（翻倍潜力）
- **日线级别密集** = 具体切入点

**仓位**：趋势确立的右侧信号，**确定性重仓**

### 实战组合（分批建仓逻辑）
1. **第一阶段**：2B结构 → 轻仓试探（左侧博弈，建底仓）
2. **第二阶段**：均线密集突破 → 重仓加码（右侧确认，主力建仓）

## 第五部分：风险管理与执行

### "三有"原则（执行铁律）

| 原则 | 含义 | 具体要求 |
|------|------|----------|
| 有预期 | 基于技术推演对走势有判断 | 如：均线即将拐头、抵扣价推演 |
| 有底线 | 明确止损位 | 跌破即走，无条件执行 |
| 有计划 | 提前规划买卖点 | 买入价、止损价、目标价全部预设 |

### 止损策略

**止损位设定**：
- **前低点**（针对2B结构）：止损设在新低 B 下方，再次跌破说明反转失败
- **筹码峰下沿**：有效击穿市场共识区域 = 支撑失效
- **关键均线**：跌破关键均线支撑位
- **逻辑证伪**：当抵扣价推演的"必由之路"未能兑现（如均线未拐头反而向下）

**止损幅度**：控制在 **10% 以内**

**核心逻辑**：止损基于"逻辑证伪"而非固定百分比。预判逻辑失效 = 必须止损。

### 卖出时机

**止盈标准**：
- **均线密集区对标**：上方历史均线密集区 = 阻力位 = 分批止盈位
- 空间 = 市场平均成本在不同时期的落差

**趋势变化判断**：
- **乖离率极值**：偏离120日EMA达50%-70% = 趋势不可持续
- **协率突变**：趋势斜率变得非常陡峭（垂直拉升）= 接近尾声
- **抵扣价反转**：股价跌破高位抵扣价，导致关键均线拐头向下 = 上涨结束

### 防守型买入（完整步骤）

1. **确认大势**：股票处于"2点钟方向"（稳步上涨，基本面配合）
2. **等待回撤**：不追高，等待自然回调
3. **锁定支撑位**：
   - 筹码峰（成交量堆积区/公允价格）
   - 抵扣价推演的均线拐头支撑点（必由之路）
4. **买入**：在支撑位买入，下跌空间有限（防守强），反弹空间巨大

## 第六部分：LEI 的失败教训

### 案例：过度相信基本面，忽视图形

**错误**：持有一只下跌的股票，自认为了解基本面，坚信"市场错了"，不仅不止损反而逢低加仓。

**结果**：后来财报暴露管理层严重决策失误（公开信息无法及时获知），股价暴跌，惨败出局，十年后都没涨回割肉价。

**教训**：
1. **图形 > 基本面预期**：走势与预期不一致时，必须尊重图形
2. **敬畏未知**：图形包含所有已知和未知信息（包括内幕、雷）。不知道为什么跌 = 有你不知道的坏事
3. **底线思维**：任何时候必须有底线，跌破底线要认输

## 第七部分：完整交易流程表

| 步骤 | 动作 | 关键标准/工具 |
|------|------|---------------|
| 1. 选股 | 基本面初筛 | 收入>1亿, 增速15-20%, 现金流为正, 垄断性生意 |
| 2. 择时 | 确认趋势 | 股价处于"2点钟方向"（稳步上涨） |
| 3. 等待 | 寻找买点 | 等回撤。用抵扣价推演均线拐头，或等回落至筹码峰支撑 |
| 4. 计划 | 制定策略 | 预期目标（下一均线密集区）+ 底线（止损位）+ 盈亏比(>2:1) |
| 5. 执行 | 买入与风控 | 在"必由之路"买入。跌破底线无条件止损 |
| 6. 持有 | 趋势跟踪 | 监控协率、抵扣价、乖离率 |
| 7. 卖出 | 止盈/止损 | 到达上方密集区止盈，或乖离率极端，或逻辑证伪止损 |

## 第八部分：软件实现要点

### 需要实现的功能模块

#### 1. 数据获取模块
- 股票价格数据 (OHLCV) — 日线 + 周线
- 财务数据 (营收、利润、现金流)
- 成交量分布数据 (Volume Profile)

#### 2. 基本面分析模块
- 商业模式评估（四大条件）
- 财务指标筛选（年收入>1亿、增速15-20%、FCF>0）
- 估值计算 (P/S, P/E) — 仅供参考

#### 3. 技术分析模块
- 均线计算: MA20, MA60, EMA120（SMA + EMA 双算）
- 抵扣价计算与画框推演
- 协率计算（均线斜率变化）
- 乖离率计算（基准: 120日EMA）
- 均线密集检测（20/60/120 三线汇聚）
- 2B结构识别 + 抵扣价验证
- 筹码峰 (Volume Profile) 计算

#### 4. 信号生成模块
- 趋势方向判断（时钟方向）
- 买入信号: 2B结构(轻仓) / 均线密集突破(重仓) / 回撤至筹码峰
- 卖出信号: 均线密集区止盈 / 乖离率极端 / 抵扣价反转
- 止损/止盈价位计算
- 盈亏比计算 (>2:1)

#### 5. 风险管理模块
- 仓位管理（试探仓 vs 确认仓）
- 止损设置（前低/筹码峰下沿/逻辑证伪）
- 止损幅度监控（<10%）

#### 6. 用户界面
- K线图表 + MA20/MA60/EMA120 + 成交量
- 抵扣价推演可视化（画框）
- 筹码峰 (Volume Profile) 显示
- 信号提示（买入/卖出/止损提醒）
- 趋势变化预警

### 数据源
- **价格数据**: yfinance (Yahoo Finance)
- **财务数据**: Financial Modeling Prep API
- **实时数据**: 券商API (Alpaca, Interactive Brokers)
