# LEI 投资分析框架 - 完整研究发现

## 概述
LEI 投资分析框架是一个将基本面（战略）与技术面（战术）深度融合的系统，强调"不预测，只预言"——利用数学逻辑推演市场的"必由之路"。

---

## 第一部分：基本面分析 (Strategy: "What to Buy")

### 商业模式四大基本条件
1. **无差别产品** - 产品是标准化的，易于复制和扩张
2. **最广泛用户** - 潜在客户群体极大，面向全球
3. **自由定价权** - 价格由市场和公司决定，无政府管制
4. **市场垄断** - 在特定领域占据绝对主导地位

### 案例：Google
- 产品矩阵绝对垄断：搜索、Android、Chrome、Map、Cloud、Gmail、YouTube
- 年收入门槛：> $100M
- 财务验证：运营现金流为正 + 自由现金流为正

### 财务筛选标准
| 指标 | 门槛 | 逻辑 |
|------|------|------|
| 年收入 | > $100M | 生存门槛，跨越后进入良性发展 |
| 收入增长 | 15-20% | 最理想、最可持续的健康状态 |
| 运营现金流 | > 0 | 生意本身能赚钱 |
| 自由现金流 | > 0 | 可攻可守（研发/并购/分红/回购） |
| 运营利润率 | 越高越好 | 同业比较，坚决回避结构性亏损 |

### 估值指标使用
- **P/S 市销率** - 反映市场预期高低
- **P/E 市盈率** - 反映市场预期高低
- **核心观点**：估值不决定买卖时机，只用来理解市场预期

## 第二部分：技术分析 (Tactics: "When to Buy")

### 时钟方向比喻
| 方向 | 含义 | 状态 |
|------|------|------|
| 12-1点 | 极度看好 | 不可持续的暴涨 |
| 2点 | 最佳状态 | 基本面与股价共振，稳步上涨 |
| 3点 | 犹豫怀疑 | 横盘震荡 |
| 4点 | 不认可 | 开始下跌 |
| 5-6点 | 极度悲观 | 暴跌或深度下跌 |

**核心**：做"2点钟方向"的股票，等回调到筹码峰时买入

### 风险管理："三有"原则
1. **有预期** - 基于逻辑对股价走势有预判
2. **有底线** - 明确止损位
3. **有计划** - 规划买入点、止损点、目标位

**铁律**：当图形与基本面矛盾时，尊重图形

## 第三部分："价格/成交量/时间/空间"交易系统

### 1. 价格 (Price) - 趋势的本质是成本
- **均线 (MA/EMA)**：代表市场平均成本
- **协率 (Slope)**：均线的倾斜度，趋势方向
- **2B结构**：假跌破后迅速拉回，趋势转折信号

### 2. 时间 (Time) - 核心技术："抵扣价"
**核心原理**：若今天收盘价 > N天前的收盘价，N日均线就会向上

**"画框"推演法**：
1. 在K线图画一个框，框住未来N天
2. 查看N天前的股价走势
3. 如果抵扣价从高位掉到低位
4. 推演：只要股价站稳"必由之路"，均线必然拐头

### 3. 空间 (Space) - 利润空间
- **均线密集区 (MA Concentration)**：市场共识区域，突破后看向下一密集区
- **乖离率 (Bias Ratio)**：
  - 计算：当前收盘价与长期均线（120日EMA）的偏离度
  - 极端：50%-70%偏离 = "人性之极"，强烈回归需求
- **盈亏比**：目标空间 / 止损空间，至少2:1

### 4. 成交量 (Volume) - 分歧与共识
- 成交量大 = 分歧大（多空互搏）
- 成交量小 = 分歧小（交投清淡）
- **极端放量**：底部巨量 = 分歧产生 = 可能转折
- **筹码峰**：纵向成交量堆积 = 市场共识/公允价格

## 第四部分：核心交易信号

### 2B结构（左侧交易/抄底）
**定义**：下跌趋势中创出新低，但迅速拉回并突破前低点

**识别步骤**：
1. 找到下跌中的低点A
2. 股价跌破A创出新低B
3. 股价在B点迅速反弹，收盘价站上A点

**逻辑**：空头陷阱，下跌趋势协率改变

### 均线密集（右侧交易/趋势确认）
**定义**：短、中、长期均线汇聚在一起

**识别**：
- 均线"拧成一股绳"
- 突破后均线向上发散 = 大趋势开始

**实战组合**：
1. 第一阶段：2B结构 → 博弈性买入（左侧）
2. 第二阶段：均线密集 → 确定性买入（右侧，重仓）

## 第五部分：投资哲学

### 孙子兵法结合
| 孙子名言 | 对应 | 解决 |
|----------|------|------|
| "知可以战与不可以战者胜" | 基本面 | 值不值得买（战略） |
| "善战者胜于易胜者也" | 技术面 | 何时买卖（战术） |

### 核心原则
1. 基本面选股，技术面选时
2. 只做"2点钟方向"的趋势
3. 尊重图形，胜过尊重自己的看法
4. 管理贪婪和恐惧的心理
